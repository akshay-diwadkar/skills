# Optimize Codebase With Senior Dev

Optimize a named bottleneck, workflow, or tooling pain with evidence-backed changes that preserve behavior — planning first, implementation only on explicit request.

## When to Use

- When targeting a specific performance, build time, or CI bottleneck.
- When performing a requested repository-wide optimization sweep.

## When NOT to Use

- For broad exploratory risk discovery (use `codebase-issue-auditor`).

## Key Inputs & Outputs

- **Input**: Target bottleneck description or sweep directive.
- **Output**: Optimization assessment and evidence-backed plan.

## Bundled Validators & Tools

- `check_optimization.py`: Validates optimization measurements and behavior preservation.

## Workflow Integration

Routes optimization plans to `implement-with-senior-dev`.
