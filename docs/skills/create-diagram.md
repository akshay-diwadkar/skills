# Create Diagram

Create self-contained HTML diagrams of systems, architectures, workflows, and code relationships.

## When to Use

- When visual communication of architecture, flow, or system dependencies is required.
- When generating self-contained HTML visual diagrams for developers, managers, or stakeholders.

## When NOT to Use

- When writing raw ASCII or text-only inline charts.

## Key Inputs & Outputs

- **Input**: Architecture data JSON or structure definition.
- **Output**: Standalone, interactive HTML diagram artifact.

## Bundled Validators & Tools

- `build_diagram.py`: Compiles diagram specifications into interactive HTML.
- `validate_diagram.py`: Validates HTML structure and completeness.

## Workflow Integration

Supports `design-codebase-with-senior-dev` and `plan-with-senior-dev`.
